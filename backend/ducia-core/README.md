# ducia-core

Core traits and plugin system for the [Ducia](https://github.com/FleetingLore/ducia-listing-framework) listing framework.

Defines `DocRepository`, `AuthPlugin`, and `PluginRegistry` — no implementation, just contracts.

## Usage

```rust
use ducia_core::doc::repo::DocRepository;
use ducia_core::plugin::auth::AuthPlugin;
```
